# ITU GAE GUI
It is a PyQt5 GUI integrated with ROS2 for Autonomous II.

## Requirements 
- PyQt5
- ROS2

## Usage
First source the ROS2 then simply use the command line prompt.
```bash
source /opt/ros/[your distro]/setup.bash 
python3 AutonomousII_python.py
```
